## How to play the game

- Make sure the python library `pygame` is installed
- Execute `init.py`

## Controls (AZERTY Keyboard)

### Player 1

- `Movement`: ZQSD
- `Attack`: E
- `Jump`: A
- `Shield`: F
- `Grab`: R
- `Special`: T

### Player 2

- `Movement`: Arrow Keys
- `Attack`: RCTRL
- `Jump`: RSHIFT
- `Grab`: RETURN
- `Shield`: COLON
- `Special`: SEMI-COLON
